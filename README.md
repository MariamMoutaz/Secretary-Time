Secretary Time
Secretary Time is a simple web tool I created to help people stay focused and organize their daily tasks in one place. I combined a To-Do list with a Pomodoro timer because I found it easier to have everything on one screen instead of switching between different apps.

What it does
Stay Focused: There is a built-in timer set for 25 minutes to help you focus on one thing at a time.

Saves Your Progress: I added a feature that saves your tasks automatically in your browser. This means you can close the page or refresh it, and your list will still be there when you come back.

Easy Organizing: You can add tasks with specific times and mark them as done or save them for later. There is also a filter to help you see exactly what you need to work on.

Clean Design: I used a transparent glass-style design with a calm background to make the workspace feel less stressful.

How I built it
I used basic web technologies to build this:

HTML for the structure.

CSS for the transparent effects and the layout.

JavaScript to make the timer work and to handle saving and filtering the tasks.

How to use it
Just type in what you need to do, set a time if you want, and add it to your list. Start the timer when you're ready to work, and use the checkmark to clear tasks as you finish them.
